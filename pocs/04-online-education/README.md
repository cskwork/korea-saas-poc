# 에듀마켓 - 온라인 교육 콘텐츠 판매 플랫폼 POC

## 개요

온라인 강의 및 디지털 상품(노션 템플릿, PDF 등)을 제작·판매할 수 있는 올인원 플랫폼 POC입니다.
인프런, 클래스101 등 기존 플랫폼처럼 강의를 체계적으로 구성하고 판매할 수 있습니다.

## 주요 기능

### 1. 대시보드
- 총 수익, 이번 달 수익, 수강생 수, 게시 강의 수 한눈에 확인
- 빠른 시작 버튼 (새 강의, 디지털 상품, 수강생 관리, 수익 분석)
- 최근 강의 목록 및 인기 디지털 상품

### 2. 강의 관리 (커리큘럼 빌더)
- 강의 생성/수정/삭제
- 섹션 및 레슨 추가/삭제
- 레슨 유형: 영상, 텍스트, 퀴즈
- 무료 공개 레슨 설정
- 게시/비공개 토글

### 3. 강의 미리보기 (랜딩 페이지)
- 수강생 관점의 강의 소개 페이지
- 히어로 섹션, 학습 내용, 커리큘럼 트리
- 가격 및 수강 신청 사이드바
- 할인 배지, 수강 혜택 표시

### 4. 수강생 관리
- 수강생 목록 (이름, 등록일, 수강 강의, 진도율, 결제액)
- 평균 진도율 통계
- 프로그레스 바 시각화

### 5. 수익 분석 대시보드
- 총 수익, 월별 수익, 성장률 표시
- 월별 수익 추이 바 차트 (Canvas API)
- 강의/상품 수익 비중 도넛 차트
- 월별 상세 테이블

### 6. 디지털 상품 판매
- 노션 템플릿, PDF, 스프레드시트 등록
- 상품 카드 UI (판매량, 가격)
- 판매 링크 복사

### 7. 가격 정책
- 3단계 요금제 (무료/베이직/프로)
- 기능 비교 체크리스트
- FAQ 아코디언

## 기술 스택

- HTML5 + TailwindCSS (CDN)
- Vanilla JavaScript (ES6+)
- Canvas API (차트)
- localStorage (데이터 영속성)

## 실행 방법

```bash
# 1. 아무 HTTP 서버로 실행
npx serve .

# 2. 또는 Python 간이 서버
python3 -m http.server 8000

# 3. 또는 브라우저에서 직접 열기
open index.html
```

## 파일 구조

```
04-online-education/
├── index.html      # 메인 HTML (헤더, 네비, 모달)
├── style.css       # 커스텀 스타일
├── app.js          # 전체 로직 (SPA 라우팅, CRUD, 차트)
└── README.md       # 이 파일
```

## 데이터 초기화

최초 실행 시 데모 데이터가 자동 생성됩니다:
- 강의 3개 (React, 피그마, 파이썬)
- 수강생 8명
- 디지털 상품 4개
- 6개월 수익 데이터

데이터를 초기화하려면 브라우저 콘솔에서:
```js
localStorage.clear();
location.reload();
```
