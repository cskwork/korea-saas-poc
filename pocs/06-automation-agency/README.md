# 06-automation-agency: AutoMate Pro (업무 자동화 대행 서비스)

## 개요

중소기업의 반복적 업무를 자동화해주는 대행 서비스 POC입니다.
Make(Integromat), Zapier, n8n, Google Apps Script 등 노코드/로코드 도구를 활용하여
구독형 유지보수 모델로 월 최대 1,500만원 수익을 목표로 합니다.

## 실행 방법

```bash
# 방법 1: Python 내장 서버
cd pocs/06-automation-agency
python3 -m http.server 8006

# 방법 2: Node.js (npx)
npx serve -p 8006

# 방법 3: VS Code Live Server 등 사용
# index.html을 브라우저에서 직접 열어도 동작합니다
```

브라우저에서 `http://localhost:8006` 접속

## 주요 기능

### 1. 📊 대시보드
- 전체 운영 현황 요약 (진행 중 프로젝트, 유지보수 고객, 월 수익, 견적 수)
- 프로젝트 상태별 분포 차트
- 인기 자동화 패키지 TOP 5
- 최근 프로젝트 목록

### 2. 📦 자동화 솔루션 카탈로그
- 10개 자동화 패키지 (데모 데이터)
- 업종별 필터 (제조, 유통, 서비스, IT, 기타)
- 유형별 필터 (데이터처리, 커뮤니케이션, 리포팅, 결제정산)
- 패키지 상세 모달 (도구, 절감 시간, 가격)
- 카탈로그에서 바로 견적에 추가

### 3. 💰 ROI 계산기
- 슬라이더 + 숫자 입력 방식
- 입력: 주당 업무 시간, 시간당 인건비, 자동화 가능 비율, 투자 비용
- 실시간 결과: 월/연 절감 시간, 월/연 절감 비용, ROI%, 투자 회수 기간
- 누적 절감 효과 막대 차트

### 4. ⚙️ 워크플로우 시각화 빌더
- 트리거/액션/조건 노드 추가
- 드래그 앤 드롭 배치 (마우스 + 터치 지원)
- SVG 기반 노드 연결선 시각화
- 3개 워크플로우 템플릿 (이메일 자동응답, 주문→배송, 엑셀→보고서)
- 노드 삭제 및 캔버스 초기화

### 5. 📋 프로젝트 관리
- 6개 데모 프로젝트 (다양한 상태)
- 프로젝트 CRUD (생성, 조회, 수정, 삭제)
- 상태별 필터 (대기, 분석, 개발, 테스트, 배포, 유지보수)
- 진행률 시각화
- 패키지 연동

### 6. 📝 견적 생성기
- 자동화 패키지 체크리스트 선택
- 복잡도 선택 (단순 0.7x, 보통 1.0x, 복잡 1.5x)
- 실시간 견적 미리보기
- 견적서 저장 (localStorage)
- 텍스트 견적서 다운로드

### 7. 💎 가격 정책
- 3단계 플랜: 기본(49만), 프로(99만), 엔터프라이즈(199만)
- 월간/연간 결제 토글 (연간 20% 할인)
- 상세 기능 비교표
- CTA 버튼

## 기술 스택

- **HTML5** - 시맨틱 마크업
- **CSS3** - 커스텀 스타일 (Tailwind 없이 자체 디자인 시스템)
- **Vanilla JavaScript** - IIFE 패턴, 모듈화
- **localStorage** - 데이터 영속성
- **SVG** - 워크플로우 연결선 렌더링

## 데이터 구조

- `AutomationPackage` - 자동화 패키지 (카탈로그)
- `Project` - 고객 프로젝트
- `Quote` - 견적서
- `WorkflowNode` - 워크플로우 노드
