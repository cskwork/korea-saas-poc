# 스마트셀러 - 네이버 스마트스토어 위탁판매 관리 도구

네이버 스마트스토어에서 위탁판매(드롭쉬핑)를 운영하는 셀러를 위한 올인원 관리 도구 POC입니다.
도매매/도매꾹 등 도매 플랫폼에서 상품을 소싱하고, AI로 상품 설명을 생성하며, 주문·수익·키워드를 한 곳에서 관리할 수 있습니다.

## 주요 기능

### 📦 상품 소싱 대시보드
- 도매매/도매꾹 도매처별 상품 필터링
- 카테고리별 필터 (패션, 뷰티, 생활/주방, 전자기기, 식품)
- 최소 마진율 슬라이더로 수익성 높은 상품만 조회
- 매입가·판매가·예상 마진율·순수익 한눈에 확인
- 상품 상세 모달 (네이버 수수료 자동 반영)

### ✨ AI 원클릭 상품 등록
- 원본 상품명 기반 네이버 SEO 최적화 상품명 자동 생성
- 카테고리별 맞춤형 상세 설명 자동 생성
- SEO 키워드 및 해시태그 자동 추출
- 등록된 상품 목록 관리

### 🛒 주문 관리
- 주문 상태별 현황 대시보드 (신규주문/처리중/배송중/배송완료)
- 주문 상태 필터 및 상태 진행 처리
- 테스트 주문 생성 기능
- 주문 취소 처리

### 🧮 수익 계산기
- 카테고리별 네이버 수수료율 자동 적용 (3.5%~7.0%)
- 매입가, 판매가, 배송비, 월 판매수량 기반 수익 분석
- 건당 순수익, 마진율, 월간 예상 수익 계산
- 비용 구조 시각화 (비율 바 차트)
- 일괄 계산 내역 기록

### 🔍 네이버 SEO 키워드 리서치
- 키워드별 월간 검색량, 경쟁도, 트렌드 분석
- 연관 키워드 추천 및 추천도 표시
- 키워드 저장 기능

### 📊 매출 분석 대시보드
- KPI 카드 (총 매출, 주문수, 순수익, 평균 마진율)
- 매출 추이 및 주문 수 추이 라인 차트
- 카테고리별 매출 비중 파이 차트
- 베스트셀러 TOP 5 랭킹

## 실행 방법

별도 설치 없이 `index.html` 파일을 브라우저에서 열면 바로 사용할 수 있습니다.

```bash
# 방법 1: 파일을 직접 열기
open index.html

# 방법 2: 간단한 로컬 서버 실행
npx serve .
# 또는
python3 -m http.server 8000
```

## 기술 스택

| 구분 | 기술 |
|------|------|
| 마크업 | HTML5 |
| 스타일 | CSS3 + TailwindCSS (CDN) |
| 스크립트 | Vanilla JavaScript (ES6+) |
| 차트 | Chart.js 4.4 (CDN) |
| 폰트 | Noto Sans KR (Google Fonts) |
| 상태 관리 | localStorage 기반 클라이언트 상태 저장 |

외부 빌드 도구나 프레임워크 없이 순수 HTML/CSS/JS로 구현되었습니다.

## 비즈니스 모델

이 도구가 지원하는 비즈니스 모델은 **네이버 스마트스토어 위탁판매(드롭쉬핑)**입니다.

1. **소싱**: 도매매, 도매꾹 등 국내 도매 플랫폼에서 상품을 발굴합니다.
2. **등록**: 도매 상품을 네이버 스마트스토어에 마진을 붙여 등록합니다. AI가 SEO 최적화된 상품명과 설명을 자동 생성하여 검색 노출을 극대화합니다.
3. **판매**: 네이버 쇼핑 검색을 통해 고객이 상품을 주문합니다.
4. **배송**: 주문이 들어오면 도매처에 발주하여 고객에게 직접 배송합니다. 재고 부담 없이 운영할 수 있습니다.
5. **수익**: 판매가에서 매입가와 네이버 수수료(카테고리별 3.5%~7.0%)를 제외한 금액이 순수익입니다.

### 카테고리별 네이버 수수료율

| 카테고리 | 수수료율 |
|---------|---------|
| 전자기기 | 3.5% |
| 뷰티 | 4.0% |
| 기타 | 5.0% |
| 패션의류/잡화 | 5.5% |
| 생활/주방 | 6.0% |
| 식품 | 7.0% |

## 참고 사항

- 이 프로젝트는 POC(Proof of Concept)로, 실제 API 연동 없이 목업 데이터로 동작합니다.
- 모든 데이터는 브라우저 localStorage에 저장되며, 브라우저 데이터 초기화 시 삭제됩니다.
- 모바일 반응형 디자인이 적용되어 있습니다.
