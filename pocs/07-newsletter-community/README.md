# 07-newsletter-community: 유료 뉴스레터 & 커뮤니티 POC

## 개요
유료 뉴스레터와 커뮤니티를 운영하여 구독 수익, 스폰서십, 멤버십 수익을 창출하는 제로 자본 SaaS 플랫폼의 POC(Proof of Concept)입니다.

## 주요 기능

### 1. 📊 대시보드
- KPI 카드 (총 구독자, 유료 구독자, 월 수익, 오픈율)
- 최근 발송 뉴스레터 목록
- 구독자 구성 비율 차트
- 유료 전환율 추이

### 2. ✏️ 뉴스레터 작성/발송
- 제목, 카테고리, 본문 작성
- 발송 대상 티어 선택 (전체/무료/유료/베이직/프로)
- 스폰서 광고 삽입
- 즉시 발송 / 임시 저장 / 예약 발송
- 임시저장 불러오기/삭제

### 3. 👥 구독자 관리
- 구독자 목록 (이름, 이메일, 티어, 상태, 가입일)
- 티어별/상태별 필터링 및 검색
- 구독자 추가/삭제
- 무료→베이직, 베이직→프로 업그레이드

### 4. 💬 커뮤니티 게시판
- 카테고리별 필터 (토론/Q&A/공지)
- 게시글 작성
- 댓글 기능
- 시간순 정렬

### 5. 💰 수익 대시보드
- KPI (MRR, ARPU, 목표 달성률, 이탈률)
- 월별 수익 추이 차트 (구독료/스폰서/멤버십)
- 수익원별 비율 분석
- 목표 달성 현황 (매출, 유료 구독자, 오픈율)

### 6. 📚 뉴스레터 아카이브
- 발송 완료 뉴스레터 그리드 뷰
- 프리미엄 콘텐츠 태그
- 클릭 시 전체 미리보기 모달

### 7. 💎 구독 플랜 가격 페이지
- 3단계 플랜 비교 (무료/베이직/프로)
- 플랜별 혜택 목록
- FAQ 섹션

## 기술 스택
- **HTML5** + **CSS3** + **Vanilla JavaScript**
- **TailwindCSS** (CDN)
- **Noto Sans KR** (Google Fonts)
- **localStorage** (데이터 영속성)

## 실행 방법

### 방법 1: 직접 열기
```
브라우저에서 index.html 파일을 직접 열면 됩니다.
```

### 방법 2: 로컬 서버
```bash
# Python
python3 -m http.server 8080

# Node.js
npx serve .

# 이후 브라우저에서 http://localhost:8080 접속
```

## 데이터 구조
모든 데이터는 localStorage에 `newsletter-community-poc` 키로 저장됩니다.

- **newsletters**: 뉴스레터 목록 (제목, 본문, 카테고리, 발송 대상, 상태 등)
- **subscribers**: 구독자 목록 (이름, 이메일, 티어, 상태, 가입일 등)
- **communityPosts**: 커뮤니티 게시글 (제목, 내용, 카테고리, 댓글 등)
- **revenue**: 월별 수익 데이터 (구독료, 스폰서, 멤버십)
