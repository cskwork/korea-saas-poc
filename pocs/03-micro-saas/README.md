# 📅 예약잇다 (YeyakItda) - POC

한국 소상공인을 위한 예약 관리 마이크로 SaaS 프로토타입

## 🎯 개요

미용실, 네일샵, 병원, 카페 등 예약 기반 소상공인을 위한 올인원 예약 관리 시스템입니다.
수기 예약장을 디지털로 전환하여 노쇼 감소, 고객 관리 자동화를 목표로 합니다.

## 🚀 실행 방법

```bash
# 방법 1: 파일 직접 열기
open index.html

# 방법 2: 로컬 서버
python3 -m http.server 8080
# → http://localhost:8080

# 방법 3: VS Code Live Server
# index.html 우클릭 → Open with Live Server
```

## 📱 주요 기능

### 1. 대시보드 (📊)
- 오늘의 예약 현황 (전체/확정/대기/취소)
- 주간 예약 통계 차트
- 오늘 예약 타임라인
- 예약 상태 변경 (대기 → 확정/취소)

### 2. 예약 관리 (📆)
- 월간 캘린더 뷰
- 날짜별 예약 건수 표시 (도트)
- 날짜 선택 시 예약 목록
- 새 예약 추가 모달

### 3. 고객 관리 (👥)
- 고객 목록 (방문 횟수순 정렬)
- 실시간 검색 (이름/연락처)
- 고객 상세 정보 및 방문 이력

### 4. 알림톡 미리보기 (💬)
- 카카오톡 스타일 UI
- 예약 확인 알림
- 리마인더 알림
- 취소 안내 알림

### 5. 고객 예약 페이지 (🌐)
- 3단계 예약 플로우 (서비스→날짜/시간→정보입력)
- 실시간 예약 가능 시간 표시
- 예약 완료 확인

### 6. 요금제 (💳)
- Free / Pro (₩30,000) / Business (₩80,000)
- 기능 비교표

## 🛠 기술 스택

- HTML5 / CSS3 / Vanilla JavaScript
- TailwindCSS (CDN)
- localStorage (데이터 저장)

## 📂 파일 구조

```
03-micro-saas/
├── index.html    # 메인 HTML (전체 페이지 구조)
├── style.css     # 커스텀 스타일
├── app.js        # 앱 로직 (데이터, 렌더링, 이벤트)
└── README.md     # 이 파일
```

## 📋 데이터 모델

- **Reservation**: 예약 정보 (고객명, 연락처, 날짜, 시간, 서비스, 상태)
- **Customer**: 고객 정보 (이름, 연락처, 방문 횟수, 마지막 방문일)
- **Business**: 매장 정보 (상호, 서비스 목록, 영업 시간)

## 🎨 디자인 특징

- 모바일 퍼스트 반응형 (320px ~ 1440px)
- 한국어 UI 전체 적용
- 카카오 스타일 알림톡 미리보기
- 상태별 색상 코드 (확정=초록, 대기=노랑, 취소=빨강)

## 📝 참고

- POC 단계로 모든 데이터는 localStorage에 저장됩니다
- 첫 실행 시 샘플 데이터가 자동 생성됩니다
- 데이터 초기화: 브라우저 개발자 도구 → Application → Local Storage → 삭제
