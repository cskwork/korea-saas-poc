# 니치 커뮤니티 플랫폼 POC

## 개요
특정 관심사/직업군을 타겟으로 한 프리미엄 커뮤니티 플랫폼의 POC(Proof of Concept)입니다.

## 실행 방법
1. `index.html`을 브라우저에서 직접 열기
2. 또는 로컬 서버 실행: `python3 -m http.server 8080`

## 기능
- 커뮤니티 피드 (게시글/댓글/좋아요)
- 채널/카테고리 관리
- 멤버 프로필 & 뱃지 시스템
- 유료 멤버십 관리 (무료/프리미엄)
- 관리자 대시보드 (회원수, 활동, 수익)
- 멤버십 가격 정책 페이지

## 기술 스택
- Vanilla HTML/CSS/JS
- TailwindCSS (CDN)
- localStorage (데이터 저장)

## 데모 계정
- 관리자: 자동 로그인 (김관리)
- 일반 회원 데모 데이터 포함

## 수익 모델

- 광고 수익
- 제휴 수익
