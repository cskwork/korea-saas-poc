# 링크잇 - 제휴마케팅 & 콘텐츠 커머스 플랫폼

## 📋 개요

**링크잇**은 제로 자본으로 시작하는 제휴마케팅 & 콘텐츠 커머스 올인원 SaaS 플랫폼입니다.
쿠팡 파트너스, 네이버 애드포스트, 텐핑 등 주요 제휴 프로그램을 통합 관리하고,
블로그/SNS 콘텐츠를 통해 제휴링크 수익을 극대화합니다.

- **목표 수익**: 월 최대 2,000만원
- **핵심 채널**: 쿠팡 파트너스, 네이버 애드포스트, 텐핑
- **콘텐츠 유형**: 비교 리뷰, 큐레이션 커머스, 추천 리스트

## 🚀 실행 방법

### 방법 1: 직접 열기
```
브라우저에서 index.html 파일을 직접 열기
```

### 방법 2: 로컬 서버
```bash
# Python 3
python3 -m http.server 8080

# Node.js
npx serve .
```

브라우저에서 `http://localhost:8080` 접속

## 🎯 주요 기능

### 1. 📊 대시보드
- 이번 달 수익/클릭/전환 총괄 통계
- 일별 수익 추이 라인 차트
- 프로그램별 수익 비율 도넛 차트
- 인기 제휴 링크 TOP 5
- 월 목표 달성률 프로그레스 바

### 2. 🔗 제휴 링크 관리
- 링크 CRUD (등록/조회/수정/삭제)
- 프로그램별 필터링 (쿠팡/네이버/텐핑/기타)
- 카테고리 분류
- 상태 관리 (활성/비활성/만료)
- 단축 URL 자동 생성

### 3. 📈 클릭/전환 추적 분석
- 오늘 실시간 통계
- 월별 수익 비교 바 차트
- 시간대별 클릭 분포
- 카테고리별 성과 분석

### 4. 📝 콘텐츠 템플릿
- **비교 리뷰**: 2~4개 제품 비교 테이블 자동 생성
- **추천 리스트**: TOP N 순위 리스트
- **상세 리뷰**: 장단점 분석 단일 제품 리뷰
- HTML 코드 생성 및 클립보드 복사

### 5. 📱 SNS 포스팅 생성기
- 쿠팡 스타일 제품 카드
- 네이버 스타일 제품 카드
- 인스타그램 스타일 제품 카드
- 해시태그 자동 생성
- SNS 텍스트 및 카드 HTML 복사

### 6. 🏢 제휴 프로그램 비교
- 쿠팡파트너스 vs 네이버 애드포스트 vs 텐핑
- 수수료율, 정산 주기, 최소 정산 금액 비교
- 장단점 분석
- 카테고리별 추천 프로그램

## 🛠 기술 스택

- **HTML/CSS/JS**: Vanilla (프레임워크 없음)
- **TailwindCSS**: CDN 방식
- **차트**: Canvas API 자체 구현
- **데이터 저장**: localStorage
- **폰트**: Noto Sans KR (Google Fonts)


## 💡 데모 데이터

최초 실행 시 15개의 데모 제휴 링크와 30일간의 시뮬레이션 데이터가 자동 생성됩니다.
데이터는 localStorage에 저장되어 브라우저를 닫아도 유지됩니다.

데이터를 초기화하려면 브라우저 콘솔에서:
```javascript
localStorage.removeItem('linkit_data');
location.reload();
```

## 📱 반응형 디자인

- **데스크톱**: 2열 레이아웃, 4열 통계 그리드
- **태블릿**: 2열 통계 그리드, 2열 프로그램 비교
- **모바일**: 1열 레이아웃, 햄버거 메뉴

## 📊 수익 모델

| 프로그램 | 평균 수수료율 | 월 목표 클릭 | 예상 전환율 | 월 예상 수익 |
|---------|------------|-----------|----------|-----------|
| 쿠팡 파트너스 | 3~7% | 50,000 | 2~5% | 500만~1,000만원 |
| 네이버 애드포스트 | CPC 기반 | 100,000 | - | 300만~500만원 |
| 텐핑 | 5~15% | 30,000 | 3~8% | 200만~500만원 |

## ⚠️ 제한사항

- 실제 API 연동 없음 (시뮬레이션 데이터)
- 클릭/전환 추적은 데모용 랜덤 데이터
- SNS 카드는 HTML 기반 (실제 이미지 다운로드 미지원)
- localStorage 용량 제한 (5MB 이하)
