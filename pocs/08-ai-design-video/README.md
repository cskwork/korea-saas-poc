# 🎨 크리에이트잇 - AI 디자인 & 영상 제작 대행 POC

## 개요

AI 도구(Canva, Midjourney, CapCut 등)를 활용한 디자인 및 영상 제작 대행 서비스의 프로토타입입니다.
제로캐피탈(무자본)으로 시작 가능한 한국 시장 타겟 SaaS 모델입니다.

## 목표 수익

- **월 최대 1,500만원** (패키지 단건 + 월 구독 모델)

## 주요 기능

| 기능 | 설명 |
|------|------|
| 📊 주문 대시보드 | 전체 주문 현황, 통계, 빠른 작업 |
| 📋 주문 관리 | 주문 생성/수정/삭제, 상태별 필터링 |
| 🔄 워크플로우 | 의뢰접수 → 시안작업 → 수정 → 납품 관리 |
| 🖼️ 포트폴리오 | 카테고리별 작업 샘플 갤러리 |
| 🤖 AI 도구 비교 | 무료/유료 AI 도구 추천 및 비교 매트릭스 |
| 💰 수익 분석 | 월별 매출 차트, 카테고리 비중, 목표 달성률 |
| 💎 가격표 | 단건/구독 패키지 비교 및 주문 연동 |

## 기술 스택

- HTML5 / CSS3 / Vanilla JavaScript
- TailwindCSS (CDN)
- Canvas API (차트)
- localStorage (데이터 영속성)

## 실행 방법

```bash
# 로컬 서버로 실행 (아무거나 선택)
npx serve .
# 또는
python3 -m http.server 8080
# 또는
직접 index.html 파일을 브라우저에서 열기
```

## 파일 구조

```
08-ai-design-video/
├── index.html    # 메인 SPA (전체 페이지 구조)
├── style.css     # 스타일시트 (Tailwind + 커스텀)
├── app.js        # 앱 로직 (네비게이션, CRUD, 차트)
└── README.md     # 이 파일
```

## 데모 데이터

첫 실행 시 자동으로 5건의 샘플 주문이 생성됩니다:
- 김마케팅: SNS 썸네일 (납품완료)
- 이커머스: 상세페이지 (시안작업중)
- 박유튜브: 영상 편집 (시안완료)
- 최스타트업: 로고 디자인 (수정중)
- 정쇼핑몰: 종합 패키지 (의뢰접수)

데이터를 초기화하려면 브라우저 개발자 도구에서:
```javascript
localStorage.clear();
location.reload();
```
