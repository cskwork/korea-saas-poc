# ✍️ AI 콘텐츠 대행 서비스 (POC)

한국 중소기업(SMB)을 위한 AI 기반 콘텐츠 대행 서비스 프로토타입입니다.

## 📋 개요

- **타겟**: 마케팅 리소스가 부족한 한국 중소기업
- **핵심 가치**: AI를 활용하여 블로그, 상품 설명, 광고 카피를 빠르고 합리적으로 제작
- **형태**: 단일 페이지 애플리케이션 (SPA) POC

## 🚀 주요 기능

| 기능 | 설명 |
|------|------|
| **AI 콘텐츠 생성** | 주제 입력 시 블로그/상품설명/광고카피 자동 생성 (데모) |
| **포트폴리오** | 업종별 콘텐츠 제작 사례 갤러리 (카테고리 필터링) |
| **요금제** | 스타터/프로/엔터프라이즈 3단계 플랜 |
| **대시보드** | 주문 현황 관리 및 통계 (데모 데이터) |

## 📁 파일 구조

```
01-ai-content-agency/
├── index.html   # SPA 셸 (TailwindCSS CDN, 모바일 반응형)
├── style.css    # 커스텀 스타일 (애니메이션, 스크롤바, 카드 효과)
├── app.js       # 앱 로직 (네비게이션, 콘텐츠 생성, 포트폴리오, 대시보드)
└── README.md    # 프로젝트 문서
```

## 💻 실행 방법

별도 빌드 없이 `index.html`을 브라우저에서 열면 됩니다.

```bash
# 로컬 서버로 실행 (선택)
npx serve .
# 또는
python3 -m http.server 8000
```

## 🛠 기술 스택

- **HTML5** — 시맨틱 마크업, 한국어(ko) 설정
- **TailwindCSS** (CDN) — 유틸리티 기반 스타일링
- **Vanilla JavaScript** — 프레임워크 없는 순수 JS
- **모바일 퍼스트** — 반응형 디자인 (320px~)

## 📌 참고 사항

- 이 프로젝트는 POC(Proof of Concept)이며, 실제 AI API 연동 없이 템플릿 기반으로 콘텐츠를 시뮬레이션합니다.
- 모든 데이터(포트폴리오, 주문 현황)는 클라이언트 측 더미 데이터입니다.
- 프로덕션 전환 시 백엔드 API, 사용자 인증, 결제 시스템 연동이 필요합니다.
